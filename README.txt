A seamless payment gateway for bangladeshi MFS

== Description ==

PayWave is a versatile plugin designed to integrate all Bangladeshi payment gateways seamlessly. It simplifies online transactions by offering a unified platform for various payment methods, ensuring secure and efficient processing. Ideal for businesses, PayWave supports popular gateways like bKash, Nagad, Rocket, and more, enhancing the e-commerce experience.

== Installation ==

* Upload the PayWave plugin to `/wp-content/plugins/` directory or Install it from Plugins > Add New Plugin and search by "PayWave", then install the plugin via wordpress dashboard.
* Activate PayWave through the 'Plugins' menu in WordPress

== Usage ==

Go to WooCommerce > Payments area to configure PayWave with your woocommerce store.
